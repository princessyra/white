Introduction to Attributes

 The name is the property you want to set. For example, the paragraph p element in the example carries an attribute whose name is align, which you can use to indicate the alignment of paragraph on the page.
 The value is what you want the value of the property to be set and always put within quotations. The below example shows three possible values of align attribute: left, center and right.		 Attributes

HTML Code

            <!DOCTYPE html>
            <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>HTML Attributes Example</title>
            </head>
            <body>
                <a href="https://www.example.com" target="_blank">Visit Example</a><br>
            
                <img src="https://www.example.com/image.jpg" alt="Example Image" style="width: 200px;"><br>
            
                <p style="color: blue; font-size: 20px;">This is a blue paragraph with larger text.</p><br>
            
                <input type="text" placeholder="Enter your name" value="John Doe"><br>
            
                <button disabled>Submit</button>
            </body>
            </html>
