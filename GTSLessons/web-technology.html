Web Technology

HTML: The Structure of the Web

Markup Language:

HTML (HyperText Markup Language) is the foundation of all websites. It provides the structure and basic content elements like headings, paragraphs, links, and images.

Document Structure:

HTML documents follow a basic structure, with elements such as head for metadata and body for content visible to users.

Elements and Tags:

HTML uses elements encapsulated in tags to define content. Tags like h1, p, and a specify the type of content and its presentation.

CSS: Styling the Web

Separation of Content and Design:

CSS (Cascading Style Sheets) allows web developers to style HTML elements separately from the structure. It controls the layout, color, font, and overall visual presentation.

Selectors and Properties:

CSS uses selectors to target HTML elements and apply styles. Properties like color, margin, and font-size define the appearance of the selected elements.

Responsive Design:

CSS enables responsive design, allowing websites to adapt to different screen sizes. Techniques like media queries ensure that web pages look good on both desktop and mobile devices.
